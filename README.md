# Quiz Application using Python

A console-based Quiz Application built using Python.  
The application randomly selects 5 questions per run with **no repetition** across multiple attempts until all questions are exhausted.

## Features
- Random questions every run
- No repetition until full cycle completes
- Score calculation
- Replay option
- Single-file Python project

## Technologies Used
- Python
- Random module
- Data Structures

## How to Run
```bash
python quiz_app.py
